## Installation and running code 
- Install `uv`
- **Run** `uv sync`, `uv lock --upgrade`
- **Run** `uv run marimo edit` **insert file name**.py for marimo notebooks 
- **Run** `uv run jupyter lab` for ipynb notebooks
